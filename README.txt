README - Juego "The Clash of the Guardians"

Código del Juego
El código proporcionado es un juego de cartas llamado "The Clash of the Guardians". El juego involucra la creación y selección de guardianes, el combate entre dos jugadores, y la administración de cartas en mano y en el campo de batalla. El código incluye las siguientes características:

Funciones

-Creación de guardianes con atributos como nombre, tipo, puntos de vida (PV), puntos de ataque (PA) y puntos de defensa (PD).
-Carga de cartas desde un archivo Cartas.txt al comienzo del juego.
-Revuelto de las cartas cargadas para asegurar que las cartas se barajen aleatoriamente.
-Selección de cartas iniciales para cada jugador.
-Combate entre dos jugadores, donde los jugadores seleccionan cartas, atacan y administran sus puntos de vida.
-Creación de nuevos guardianes por parte del jugador.
-Visualización de los guardianes creados por el jugador.
-Menú principal que permite iniciar el juego, mostrar el historial de partidas, crear nuevos guardianes y salir del juego.
-Instalación de Visual Studio Code y Compilador
-Para ejecutar este juego, sigue estos pasos:

-Instalación de Visual Studio Code:
-Descarga el instalador adecuado para tu sistema operativo (Windows, Mac o Linux).
-Sigue las instrucciones de instalación proporcionadas por el asistente.
-Compilador (GCC o MinGW) (Windows):

Si estás utilizando Windows, es posible que necesites un compilador como MinGW. Asegúrate de instalar MinGW antes de compilar y ejecutar el juego. Puedes descargar MinGW desde su sitio oficial 

¡¡¡¡¡IMPORTANTE!!!!!
Asegúrate de tener un archivo llamado Cartas.txt que contenga las cartas necesarias para el juego. Este archivo se utiliza para cargar las cartas al comienzo del juego. Asegúrate de que el archivo esté en el mismo directorio que el código del juego.
Una vez que hayas instalado Visual Studio Code y, si es necesario, el compilador, y tengas el archivo Cartas.txt, podrás ejecutar el juego desde Visual Studio Code.

Ejecución del Juego
Después de seguir los pasos de instalación, abre Visual Studio Code, carga el código del juego y ejecútalo. El menú principal te permitirá comenzar el juego, crear nuevos guardianes, mostrar los guardianes creados y salir del juego.
